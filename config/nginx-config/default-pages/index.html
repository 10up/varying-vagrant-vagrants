If you see this page, the nginx web server is successfully installed and working but without settings.

This could be because of:

  • The vagrant provision didn't finished yet.
  • Provisioning failed, you can find the logs in VVV/log/provisioners.
  • You didn't update rightly VVV, use git pull and try a new vagrant provision.

Reprovision using vagrant up --provision or wait for the command to finish. If you still have problems get in touch on GitHub or Slack
